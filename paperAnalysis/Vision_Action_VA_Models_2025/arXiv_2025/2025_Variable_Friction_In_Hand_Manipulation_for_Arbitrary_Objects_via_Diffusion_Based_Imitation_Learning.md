# Variable-Friction In-Hand Manipulation for Arbitrary Objects via Diffusion-Based Imitation Learning

> [!abstract] **Quick Links & TL;DR**
> - **Links**: [arXiv](https://arxiv.org/abs/2503.02738) · [Project/Code](https://sites.google.com/view/vf-ihm-il/home)
> - **Summary**: 这篇论文把可变摩擦手的在手操控，从“按几何手工写规则/写奖励”改成“RL生成平滑示范 + hindsight重标注 + 扩散模仿学习 + 仿真/真实共训”，从而把操控对象从规则多边形扩展到非多边形，并支持任意目标位姿。
> - **Key Performance**: 真实5物体平均成功率 **71.3%**；成功样本平均误差 **2.676 mm / 1.902°**

> [!info] **Agent Summary**
> - **task_path**: 当前关节状态 + 物体SE(2)位姿 + 目标SE(2)位姿 -> 混合离散/连续摩擦-运动动作 -> 物体在手内达到目标位姿
> - **bottleneck**: VF手的非完整接触动力学让手工规划只覆盖少数规则几何，而逐物体奖励工程又无法扩展到任意形状与任意目标
> - **mechanism_delta**: 用“平滑但不必精确”的RL轨迹做覆盖，再经HER重标注转成目标条件示范，最后用扩散策略在仿真+少量真实数据上共训来学多模态接触转换
> - **evidence_signal**: 五物体真实实验 + co-train相对sim-only/real-only的消融提升 + Cube上相对RL/A*的对比
> - **reusable_ops**: [HER示范重标注, 仿真-真实等概率混合采样]
> - **failure_modes**: [Three-Cylinder上本应滑动却发生滚动, 累积视觉/执行误差导致10步预算内未收敛]
> - **open_questions**: [能否训练跨对象共享的单一通用策略, 能否通过可调掌宽或触觉反馈消除曲面物体受力失配]

## Part I：问题与挑战

这篇论文解决的是**低自由度可变摩擦手（VF hand）如何对任意形状物体做高精度在手操控**的问题。

### 1) 真正难点是什么？

不是“能不能动起来”，而是：

1. **接触模式切换高度微妙**  
   VF手靠“高摩擦抓住 / 低摩擦滑动”切换来实现滚动和滑移。对象运动不是简单欧氏直线，而是受接触几何、摩擦状态、手指相对方向共同制约的**非完整运动**。

2. **传统方法的瓶颈在于规则写死**  
   之前的A*+手工primitive方法，只适用于：
   - 规则多边形；
   - 从一个平面滚到另一个平面；
   - 目标位姿必须落在“某个平面贴住手指”的可达集合里。  
   一旦对象是星形、圆柱组合体这类非规则外形，手写规则就很难覆盖。

3. **RL也不容易直接解**  
   若想用RL做高精度控制，作者发现必须对**每个物体单独做奖励工程**，尤其要编码各自接触面和内角信息。  
   这意味着：  
   **瓶颈不是策略网络不够大，而是监督信号和目标覆盖方式不可扩展。**

### 2) 输入/输出接口

- **输入**：手指关节状态、物体当前2D位姿（x, y, yaw）、目标位姿；IL中速度项直接零填充以避免sim/real速度不一致。
- **输出**：混合动作
  - 离散部分：6种摩擦/运动模式（左右手指谁主导、谁高/低摩擦、滑动还是旋转）
  - 连续部分：主动手指的关节角增量
- **任务边界**：
  - 只做**平面SE(2)**在手操控，不是6D全姿态操作
  - 最多10个动作步
  - 依赖顶视相机 + ArUco标记估计物体位姿
  - 最终高精度策略仍是**逐物体训练**，不是零样本跨物体泛化

### 3) 为什么现在值得做？

因为VF手本身机械上很简单，远比多指灵巧手便宜、易维护。  
如果能用学习方法解除其几何限制，就有机会以低成本硬件实现相当强的在手操控能力。  
而扩散策略 + 少量真实数据共训，正好提供了一个**低数据、高多模态表达能力**的方案。

## Part II：方法与洞察

### 方法主线

论文的策略不是直接“端到端从零学精确控制”，而是分成两层：

1. **先用RL学一个平滑探索者**  
   目标不是高精度到点，而是生成**稳定、平滑、覆盖合理接触转换**的轨迹。  
   这样奖励函数就不必为每个物体精心重写。

2. **再把轨迹变成示范数据**  
   用 hindsight goal relabeling：  
   把一条轨迹的**终点**重标注成它的目标。  
   于是，哪怕RL本来不是为那个具体目标优化的，只要轨迹合理，就能成为目标条件示范。

3. **最后用扩散策略做行为克隆**  
   对每个物体，混合：
   - 10,000条仿真示范
   - 100条真实示范  
   用扩散模型学习“给定当前状态和目标，下一步该走什么动作分布”。

### 为什么这里用扩散策略而不是普通BC？

因为在手操控明显是**多模态**的：  
同一个起点和目标，可能存在多条都可行的滑动/旋转组合路径。

普通回归式BC容易把多种可行动作“平均化”，产生一个谁都不像的无效动作。  
扩散策略更适合表达这种**多峰动作分布**，因此能保留多种合理接触迁移路径。

### 关键设计

#### 1. 混合动作空间
作者没有让策略直接输出所有底层马达控制，而是把动作拆成：

- **离散高层模式**：决定当前是“在左/右指上滑”、“顺/逆时针转”等
- **连续低层幅度**：输出本次主动手指要走多大角度

这样做的因果作用是：

- 把“接触模式选择”从连续控制里显式抽出来；
- 减少搜索空间；
- 更贴合VF手天然的物理语义。

#### 2. RL只负责“可用覆盖”，不负责“最终精度”
这是本文最重要的策略转变。

作者发现：  
直接训练一个高精度RL控制器，必须深度依赖逐物体奖励设计。  
于是他们把RL的职责降级为：

- 产生足够平滑的轨迹；
- 覆盖合理的接触转移；
- 能在仿真和现实里自动产出示范。

这相当于把难问题从“优化最终精度”改成“生成可模仿的接触序列”。

#### 3. 仿真-真实共同训练，而不是先仿真后微调
作者不是先训sim policy再用少量real fine-tune，而是直接**co-train**。  
关键作用是：

- 仿真提供覆盖面；
- 真实数据提供接触/摩擦/视觉误差的落地校正；
- 两者混合能避免纯微调时对少量真实数据过拟合。

论文里甚至显式比较了 co-train vs fine-tune，co-train更稳。

### 核心直觉

**这篇论文真正改变的，不是控制器结构本身，而是“任务监督的生成方式”。**

从前：
- 需要人工枚举接触规则，或
- 需要为每个物体写精细奖励

现在：
- 先让RL提供“平滑的、物理上可行的接触迁移”
- 再用HER把这些迁移自动变成大量目标条件示范
- 再用扩散策略拟合这些多模态示范
- 用少量真实数据把仿真分布拉回真实接触分布

所以发生的因果链条是：

**手工几何约束/奖励工程减少**  
→ **目标覆盖与动作分布支持集变大**  
→ **策略学到的是“如何切换接触模式达成目标”，而不是“某个物体的专用规则”**  
→ **从规则多边形+有限目标，扩展到非规则对象+任意目标位姿。**

### 战略权衡

| 设计选择 | 解决的瓶颈 | 带来的能力 | 代价/风险 |
|---|---|---|---|
| 混合离散-连续动作 | 连续控制难以稳定表达接触模式切换 | 更稳的滑动/旋转决策，减少探索负担 | 动作语义写死在VF手结构里，可迁移性受硬件限制 |
| RL只学“平滑探索” | 逐物体高精度奖励工程难扩展 | 能快速生成跨对象示范 | 最终精度要靠后续IL弥补 |
| HER重标注示范 | 真实目标数据昂贵 | 任意轨迹终态都可转成监督信号 | 依赖轨迹本身足够平滑、在任务域内 |
| 仿真+真实共训 | sim-to-real接触偏差 | 少量真实数据下提升落地成功率和稳定性 | 仍需可用仿真器、系统辨识、domain randomization |
| 顶视视觉+AruCo观测 | 状态获取成本高 | 低成本高精度位姿监督 | 依赖标记、视野无遮挡、平面操作场景 |

## Part III：证据与局限

### 关键证据

- **[真实多物体结果]**  
  在5种对象（规则多边形、非规则多边形、非多边形）上，真实硬件平均成功率达到 **71.3%**，成功样本平均误差 **2.676 mm / 1.902°**。  
  这说明方法不是只会操控立方体，而是在给定物体训练后，能覆盖更复杂几何。

- **[共训消融]**  
  五物体平均成功率：
  - 仅仿真：**32.0%**
  - 仅真实：**54.0%**
  - 仿真+真实共训：**71.3%**  
  说明真正有效的不是“多一点数据”这么简单，而是**覆盖面（sim）+校正（real）**的组合。

- **[数据效率]**  
  在Cube Cylinder上，随着真实示范增加，性能上升并在约200条后趋于平台；  
  200条真实示范时，co-train可达 **93.3%**，明显优于真实-only的 **73.3%**。  
  这支持其“少量真实数据即可明显受益”的主张。

- **[与RL基线对比]**  
  在Cube上，最终IL策略成功率 **76.7%**，高于奖励工程RL的 **66.7%**；  
  训练时长从 **15h** 降到 **3.5h RL + 2h IL**。  
  也就是说，能力提升不是靠更重训练换来的。

- **[与A*规划对比]**  
  对Cube/Hexagon，在A*可处理的预设目标上，本文位置误差更低；更关键的是，**A*** 根本不支持任意目标位姿，而本文支持。  
  能力跃迁的本质是“可达目标集合变大”。

### 局限性

- **Fails when**: Three-Cylinder这类曲面组合几何在低摩擦滑动阶段可能触发非预期滚动；当视觉误差、接触误差和执行误差叠加时，策略可能需要额外滑动+旋转补救，仍可能在10步预算内失败。
- **Assumes**: 依赖逐物体训练的精确策略；依赖MuJoCo仿真、系统辨识、domain randomization、AruCo位姿追踪、顶视相机标记可见；资源上依赖约44核CPU做RL采样、A100做约2小时IL训练，以及约1小时/物体的真实示范采集。
- **Not designed for**: 零样本跨对象通用策略、无标记感知、6D全姿态在手操控、快速动态操作、以及无需对象特定训练的“真正任意物体”设置。

### 可复用部件

1. **HER示范合成**  
   把“次优但平滑”的轨迹转成大量目标条件示范，非常适合奖励难写的接触操控问题。

2. **混合动作因子化**  
   先选接触模式，再输出细粒度运动幅度，适合非完整、模式切换显著的机器人任务。

3. **仿真-真实等概率共训**  
   在真实数据很少时，比“纯仿真”或“先仿真后微调”更稳，适合接触动力学偏差明显的场景。

4. **对象特定精调 + 通用探索者**  
   用一个共享RL探索者产出多对象示范，再训练对象特定精确策略，是一个很务实的工程折中。
