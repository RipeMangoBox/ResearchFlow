# paperIDEAs

This directory stores downstream research outputs built on top of the
ResearchFlow knowledge base.

Typical contents include idea notes, focused plans, reviewer stress tests,
comparison writeups, and implementation-facing research roadmaps.

> **Note**: Idea outputs are also stored in the backend database (`idea_deltas` + `delta_cards` tables). This directory is a generated export. For programmatic access, use `/api/v1/graph/ideas/` or `/api/v1/search/ideas`.
