## 背景与动机

黑盒无监督域适应（BUDA）设定了一个极具挑战性的实际场景：企业或机构仅能通过 API 调用获取源模型的预测输出，既无法访问源训练数据，也无法获得源模型参数。例如，一个基于 ImageNet 训练的图像分类模型以黑盒服务形式提供，用户需要将其适应到医疗影像或工业检测等目标域，但只能拿到目标域无标注数据和源模型对它们的预测结果。

现有方法主要从三个方向应对这一挑战。**DINE** [15] 通过信息最大化和熵最小化从单个或多个黑盒预测器中提取知识，但仅依赖伪标签进行单阶段适应，容易在域差异较大时积累错误。**BETA** [16] 采用样本划分和两步蒸馏来缓解确认偏置，然而其两阶段计算密集（20247 MiB 显存、1101 s/epoch），且仍局限于像素级/浅层特征的对齐。**SEAL** [17] 和 **AEM** [19] 分别通过特征分离和对抗专家模型提升性能，但 SEAL 在 24GB GPU 上无法完成训练，AEM 需要维护两个分类器并分别更新权重，开销显著。

这些方法的共同瓶颈在于：**域适应的决策过程停留在像素级或浅层特征层面，未能充分利用生成模型深层语义理解能力进行推理层面的纠错**。具体而言，现有扩散-based DA 方法仅关注组合式生成，忽视了对文本细微差别的敏感性，导致生成图像虽多样却缺乏判别性，无法有效指导目标模型学习。此外，单阶段训练缺乏机制来识别和纠正目标模型自身的推理错误。

本文提出 RrED，将域适应从"像素对齐"提升到"推理纠错"层面，通过两阶段扩散监督框架实现黑盒设定下的高效域适应。

## 核心创新

核心洞察：**扩散模型的推理错误可以通过目标模型对"生成图像-真实图像"的差异化预测来自我识别和纠正**，因为生成图像具有更鲜明的特征和更少的干扰物体，为目标模型提供了"简化版"参考基准，从而使利用模型自身差异推理进行迭代纠错成为可能。

| 维度 | Baseline (DINE/BETA/SEAL/AEM) | 本文 RrED |
|:---|:---|:---|
| 数据流程 | 直接使用无标注目标数据+黑盒预测；无生成增强 | 两阶段：DTR 生成推理增强图像 → SRM 结合生成与真实数据自校正 |
| 架构设计 | 标准耦合编码器或独立分类器 | 解耦扩散编码器：视觉编码器+FSG 门控保留 ROI；文本编码器联合微调 |
| 训练范式 | 单阶段伪标签/蒸馏/一致性损失 | 两阶段递进：扩散监督校正 (DTR) → 差异推理纠错 (SRM) |
| 优化目标 | 最小化目标域预测误差 | Stage 1 多模态联合对齐；Stage 2 差异推理直接服务 BUDA 任务 |

## 整体框架


![Figure 2](https://researchflow.oss-cn-shanghai.aliyuncs.com/papers/f85b7bde-2cf9-4e47-8a9a-6195973b9cd0/figures/Figure_2.png)
*Figure 2 (architecture): Figure 2: Overview of the whole training of RrED. According to the BUDA setting, (a) the source model and samples generated by the source-free diffusion model are used to construct the source memory bank. (b) RrED consists of two main stages: Diffusion-driven Self-rectifying Processing (DSP) and Self-rectifying Reasoning Model (SRM). Specifically, DSP leverages the domain-shared knowledge of the diffusion model to construct positive and negative samples. Moreover, SRM corrects the reasoning errors of the target model by leveraging the middle layers.*



RrED 的整体训练流程如 Figure 2 所示，分为两个递进阶段，数据流如下：

**输入**: 无标注目标域图像 $x_t \sim \mathcal{D}_t$，以及黑盒源预测器提供的伪标签/预测分布（仅 API 访问）。

**Stage 1 — DTR (Diffusion-Target model Rectification)**:
- **解耦视觉编码器 + FSG**: 输入目标图像，通过添加 Feature-Sensitive Gate (FSG) 的视觉编码器提取噪声鲁棒特征，门控掩码 $m(x)$ 选择性保留感兴趣区域 (ROI)，非 ROI 区域池化降采样，输出"推理增强"的视觉特征。
- **联合微调文本编码器**: 输入文本提示，与视觉特征进行多模态联合对齐，在保持生成能力的同时增强对生成内容的理解。
- **扩散采样器**: 基于对齐的多模态表示，生成具有更大多样性、更鲜明特征、更少干扰物体的合成图像 $x_g \sim \mathcal{G}$。
- **目标模型初训练**: 利用生成图像 $x_g$ 及其黑盒伪标签训练目标模型，使其习得丰富语义知识。

**Stage 2 — SRM (Self-rectifying Reasoning Model)**:
- **差异推理模块**: 将目标模型同时应用于生成图像 $x_g$ 和真实目标图像 $x_t$，比较两者预测差异 $\Delta p = |p_\theta(x_t) - p_\theta(x_g)|$。
- **自校正学习**: 当预测不一致时，通过 KL 散度约束使模型在两种输入上的推理趋于一致，优先处理 BUDA 任务本身，逐步稳定特征分布边界。

**输出**: 适应后的目标模型 $p_\theta$，可直接用于目标域分类。

```
黑盒源预测器 ──→ 伪标签
                      ↓
目标图像 x_t ──→ [DTR: 解耦扩散编码器 + FSG + 文本微调]
                      ↓
              生成图像 x_g (推理增强)
                      ↓
              [SRM: 目标模型差异推理]
                 ↙            ↘
            p_θ(x_g)  ←→  p_θ(x_t)
                      ↓
              自校正损失优化
                      ↓
              最终适应模型 p_θ
```

## 核心模块与公式推导

### 模块 1: Feature-Sensitive Gate (FSG)（对应框架图 DTR 视觉编码器部分）

**直觉**: 标准扩散视觉编码器在高噪声水平下会模糊判别性区域，导致目标模型难以辨识；FSG 通过可学习门控选择性保留 ROI，使生成图像始终保持对分类任务的关键信息。

**Baseline 公式** (标准扩散视觉编码器):
$$h = f_v(x_t)$$
符号: $f_v$ = 视觉编码器, $x_t$ = 时间步 $t$ 的加噪图像, $h$ = 编码特征。

**变化点**: 标准输出对所有空间位置平等处理，噪声下 ROI 特征被淹没；FSG 引入空间选择机制，显式保留判别区域。

**本文公式（推导）**:
$$\text{Step 1: } m(x) = \sigma(W_m \cdot \text{GAP}(f_v(x)) + b_m) \quad \text{通过全局平均池化+线性层生成软门控掩码}$$
$$\text{Step 2: } \text{FSG}(f_v(x)) = f_v(x) \odot m(x) + (1 - m(x)) \cdot \text{Pool}(f_v(x)) \quad \text{ROI 保留，非 ROI 池化降采样}$$
$$\text{最终: } h_{\text{FSG}} = \text{FSG}(f_v(x_t))$$
符号: $m(x) \in [0,1]^{H \times W}$ = 空间门控掩码, $\odot$ = 逐元素乘法, $\text{Pool}$ = 空间池化, $\sigma$ = Sigmoid。

**对应消融**: 虽未在提供材料中显式给出 FSG 单独消融表，但 Figure 8 的 t-SNE 可视化显示 Stage 1 (含 FSG) 相比 Source-only 从 44.1% 提升至 79.8%，证明特征判别性的显著改善。

### 模块 2: DTR 多模态联合微调目标（对应框架图 Stage 1 优化）

**直觉**: 独立优化的视觉和文本编码器难以保证生成图像的语义一致性；联合微调使扩散模型"理解"要生成的内容，而非仅"模仿"像素分布。

**Baseline 公式** (标准扩散训练 / 伪标签适应):
$$\mathcal{L}_{\text{base}} = \mathcal{L}_{\text{cls}} + \lambda \mathcal{L}_{\text{consistency}}$$
符号: $\mathcal{L}_{\text{cls}}$ = 伪标签交叉熵, $\mathcal{L}_{\text{consistency}}$ = 一致性正则（如熵最小化）。

**变化点**: Baseline 直接优化目标模型分类性能，忽视生成过程中的跨模态对齐；DTR 将扩散监督与多模态对齐结合，让目标模型从"生成质量"中习得语义。

**本文公式（推导）**:
$$\text{Step 1: } \mathcal{L}_{\text{diff}} = \mathbb{E}_{x_0, \epsilon, t}\left[\|\epsilon - \epsilon_\theta(x_t, t, c)\|^2\right] \quad \text{标准扩散去噪损失，条件 } c \text{ 来自文本编码器}$$
$$\text{Step 2: } \mathcal{L}_{\text{align}} = -\log \frac{\exp(\text{sim}(h_v, h_t)/\tau)}{\sum_{j} \exp(\text{sim}(h_v, h_t^{(j)})/\tau)} \quad \text{视觉-文本对比对齐，} h_v=\text{FSG}(f_v(x)), h_t=f_t(c)$$
$$\text{Step 3: } \mathcal{L}_{\text{DTR}} = \mathcal{L}_{\text{diff}} + \lambda_1 \mathcal{L}_{\text{align}} + \lambda_2 \mathcal{L}_{\text{cls}}^{\text{pseudo}} \quad \text{联合优化，保留生成能力同时服务适应任务}$$
$$\text{最终: } \theta_{\text{DTR}}^* = \text{arg}\min_{\theta} \mathcal{L}_{\text{DTR}}$$
符号: $\epsilon_\theta$ = 噪声预测网络, $f_t$ = 文本编码器, $\text{sim}$ = 余弦相似度, $\tau$ = 温度系数, $\mathcal{L}_{\text{cls}}^{\text{pseudo}}$ = 黑盒伪标签分类损失。

**对应消融**: Table 7 显示 RrED Stage 1 在 VisDA-17 达到 89.4%，为 Stage 2 的 91.2% 奠定基础，两阶段协同增益 1.8%。

### 模块 3: SRM 差异推理自校正损失（对应框架图 Stage 2 优化）

**直觉**: 目标模型对"简单"生成图像和"困难"真实图像的预测差异，恰好暴露其推理盲区；利用这种差异作为自监督信号，可实现无需额外标注的错误纠正。

**Baseline 公式** (伪标签训练):
$$\mathcal{L}_{\text{pseudo}} = -\sum_{i} \hat{y}_i \log p_\theta(x_i)$$
符号: $\hat{y}_i = \text{arg}\max p_{\text{source}}(x_i)$ = 黑盒源预测伪标签。

**变化点**: 伪标签训练将源模型预测视为"ground truth"，无法识别目标模型自身的推理错误；SRM 通过模型对两种输入（$x_g$ vs $x_t$）的"自我不一致"发现错误，实现真正的自校正。

**本文公式（推导）**:
$$\text{Step 1: } \Delta p(x) = |p_\theta(x_t) - p_\theta(x_g)| \quad \text{计算同一样本对应生成/真实图像的预测差异}$$
$$\text{Step 2: } \mathbb{1}[\text{err}] = \mathbb{1}[\text{arg}\max p_\theta(x_t) \neq \text{arg}\max p_\theta(x_g)] \quad \text{识别推理错误位置}$$
$$\text{Step 3: } \mathcal{L}_{\text{SRM}} = \mathbb{E}_{x_t \sim \mathcal{D}_t, x_g \sim \mathcal{G}} \left[ \mathbb{1}[\text{err}] \cdot \mathcal{D}_{\text{KL}}(p_\theta(x_t) \| p_\theta(x_g)) + \lambda_3 \mathcal{H}(p_\theta(x_t)) \right]$$
$$\text{最终: } \theta^* = \text{arg}\min_\theta \mathcal{L}_{\text{SRM}}$$
符号: $\mathcal{D}_{\text{KL}}$ = KL 散度（使困难样本向简单样本的可靠预测靠拢）, $\mathcal{H}$ = 熵正则（保证预测置信度）, $\mathbb{1}[\text{err}]$ = 错误指示函数（仅当不一致时触发校正）。

**对应消融**: Stage 1-only (89.4%) vs. 完整两阶段 (91.2%)，SRM 贡献 +1.8% 准确率；Figure 8 显示 Stage 2 使 t-SNE 特征边界从 Stage 1 的相对分散趋于"稳定围绕最近簇中心"的紧凑分布。

## 方法谱系与知识库定位

RrED 隶属于**黑盒无监督域适应 (BUDA)** 方法谱系，直接继承自 **DINE** [15] 的两阶段思想（信息提取→适应）和 **SHOT** [11] 的源自由设定，但在四个关键 slot 上发生结构性变革：

| 父方法 | 差异 |
|:---|:---|
| **DINE** [15] | 从单阶段伪标签扩展为两阶段扩散监督；新增生成增强和差异推理 |
| **BETA** [16] | 替代计算密集的两步蒸馏为扩散生成+自校正；显存从 20247 MiB 降至 11721 MiB |
| **SEAL** [17] | 以可运行的硬件成本实现更高准确率 (91.2% vs 89.2%)，解决 SEAL 的 24GB GPU 不可行问题 |
| **AEM** [19] | 替代对抗多专家的双分类器更新为单模型差异推理，训练速度提升 3.3 倍 |
| **DDPM** [21] | 从纯生成模型改造为域适应工具：解耦编码器、添加 FSG、联合文本微调 |

**后续方向**: (1) 探索极端域差距下的扩散先验校准机制，缓解生成-目标域失配； (2) 将 FSG 门控思想推广至其他生成架构（如流模型、 consistency models）； (3) 研究 Stage 1 与 Stage 2 的交替/端到端联合优化，打破顺序依赖。

**标签**: 视觉-语言多模态 / 自监督学习 / 域适应 / 扩散模型 / 黑盒约束 / 推理纠错机制
