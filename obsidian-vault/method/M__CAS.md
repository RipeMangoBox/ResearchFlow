# CAS

CAS 是 2025 年发表于 CVPR 的动作分割数据集压缩方法，通过生成网络反演（generative network inversion）技术，将大规模动作分割数据集蒸馏为紧凑的合成数据集。该方法旨在解决动作分割任务中数据集规模大、标注成本高的问题。

值得注意的是，CAS 在另一篇 ECCV 2024 工作（Cascade Prompt Learning for Visual-Language Model Adaptation）中被引用，表明其在视觉-语言模型适配领域也具有一定的关联性和影响力。

**研究领域**: Segmentation, Dataset Distillation, Continual Learning
