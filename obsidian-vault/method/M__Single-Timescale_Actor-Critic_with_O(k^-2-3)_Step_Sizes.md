# Single-Timescale Actor-Critic with O(k^-2/3) Step Sizes

Single-timescale actor-critic converges to an ε-close globally optimal policy with sample complexity O(ε^-3), improving upon existing O(ε^-4) complexity for global optimality, by using step sizes decaying as O(k^-2/3).

**研究领域**: Reinforcement Learning
