# CMD-SE

CMD-SE是一种面向开放词汇人-物交互（Human-Object Interaction, HOI）检测的跨模态方法。该方法发表于CVPR 2024，旨在探索大型基础模型（Large Foundation Models）在开放词汇HOI检测任务中的潜力。

CMD-SE通过结合跨模态检测机制与语义嵌入技术，实现对未见过人-物交互类别的泛化检测。该方法的核心创新在于利用大规模预训练模型的丰富语义知识，突破传统HOI检测方法受限于预定义类别集合的瓶颈。

**研究领域**: Object Detection, Cross-Modal Matching, Visual Reasoning
