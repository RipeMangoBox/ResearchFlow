# Cascade Prompt Learning

CPL（Cascade Prompt Learning）是一种面向视觉-语言模型（Visual-Language Model, VLM）自适应的分层提示学习方法。该方法通过级联式的提示学习策略，在不同语义层次上逐步优化模型对下游任务的适应能力，缓解传统单层次提示学习在复杂视觉概念建模上的不足。

发表于 ECCV 2024 的 CPL 针对 CLIP 等预训练视觉-语言模型的高效迁移学习问题，提出了一种结构化的提示优化新范式。

**研究领域**: Few-Shot Learning, Domain Adaptation, Retrieval
