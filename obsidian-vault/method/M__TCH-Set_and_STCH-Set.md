# TCH-Set and STCH-Set

TCH-Set 和 STCH-Set 是 2025 年发表于 ICLR 的多目标优化方法，基于切比雪夫（Tchebycheff）标量化策略。核心创新在于"Few for Many"——用少量参考点实现多目标（many-objective）优化，解决传统方法在高维目标空间中参考点数量爆炸的问题。

STCH-Set 应为 TCH-Set 的扩展或改进变体（S 可能代表 Stochastic、Sparse 或 Structured），两者形成方法体系，共同提升多目标优化的可扩展性。

**研究领域**: Multi-Objective Optimization
