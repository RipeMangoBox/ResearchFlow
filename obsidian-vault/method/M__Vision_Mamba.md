# Vision Mamba

Vision Mamba（Vim）是一种基于双向状态空间模型的视觉表示学习方法。该方法将 Mamba 的选择性状态空间机制扩展至图像识别任务，通过双向扫描策略捕获二维空间中的全局依赖关系，同时保持与序列长度成线性关系的计算复杂度。

发表于 ICML 2024 的 Vim 与同期 VMamba 共同开创了视觉状态空间模型的新方向，为视觉骨干网络设计提供了 Transformer 和 CNN 之外的高效替代方案。

**研究领域**: Classification, Object Detection, Semantic Segmentation
