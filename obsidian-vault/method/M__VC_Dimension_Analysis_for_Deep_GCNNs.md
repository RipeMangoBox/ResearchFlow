# VC Dimension Analysis for Deep GCNNs

We derive tight upper and lower bounds on the VC dimension of deep GCNNs with ReLU activations, showing explicit dependence on depth, width, number of weights, and input dimension, and demonstrate that GCNNs can achieve the same VC dimension as standard CNNs with fewer parameters.

**研究领域**: Classification
