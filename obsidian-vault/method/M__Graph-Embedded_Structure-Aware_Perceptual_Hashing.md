# Graph-Embedded Structure-Aware Perceptual Hashing

GESAPH 是 CVPR 2025 提出的图嵌入结构感知感知哈希方法，用于神经网络知识产权保护与盗版检测。该方法将图神经网络与感知哈希技术结合，为深度模型提供类似"数字指纹"的认证机制。

随着大模型商业价值的提升，模型窃取与未授权分发成为严重问题，GESAPH 通过提取模型结构的鲁棒哈希特征，实现了对盗版模型的精准追踪与认证。

**研究领域**: Object Detection, Graph Learning
