# Optimal Test-Time Compute Scaling via Verifier-Guided Search

Optimal Test-Time Compute Scaling via Verifier-Guided Search 是一种在测试阶段动态扩展计算资源的方法，旨在提升大型语言模型的推理能力。该方法通过验证器引导的搜索策略，智能地分配额外的计算预算，而非简单地增加模型参数量。

该方法的核心思想源于 ICLR 2025 的研究，表明在推理任务中，优化测试时的计算扩展可能比单纯扩大模型规模更为有效。通过引入验证机制来评估中间推理步骤的质量，该方法能够实现计算资源的自适应分配。

**研究领域**: Math Reasoning
