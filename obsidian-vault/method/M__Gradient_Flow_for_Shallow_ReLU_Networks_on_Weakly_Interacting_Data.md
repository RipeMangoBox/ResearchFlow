# Gradient Flow for Shallow ReLU Networks on Weakly Interacting Data

For one-hidden-layer ReLU networks trained by gradient flow on n weakly correlated (nearly orthogonal) data points in high dimension, width of order log(n) suffices for global convergence with high probability, with exponential convergence rate O(1/n) and refined asymptotic behavior between O(1/n) and O(1/√n) for exactly orthogonal data exhibiting a phase transition.
