# Code Generation (代码生成)

代码生成（Code Generation）是程序合成领域的核心任务，旨在将人类意图自动转换为可执行的计算机程序。该任务涵盖从自然语言描述、部分代码片段或形式化规约出发，生成完整、正确且高效的源代码。

作为软件工程与人工智能的交叉方向，代码生成不仅要求模型理解编程语言的语法和语义，还需掌握算法设计、API 使用及软件设计模式等深层知识。近年来，大规模语言模型在该任务上取得了显著进展。
