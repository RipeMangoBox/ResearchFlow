# Agent (智能体)

Agent（智能体）是人工智能领域的核心概念之一，指能够自主感知环境、进行推理规划并执行动作以实现特定目标的系统。2026年的研究呈现出明显的趋势：智能体能力正从原子工具使用向复杂工作流编排演进，同时记忆系统、可靠性评估和长期演化能力成为关键研究方向。

当前研究特别关注智能体在真实部署场景中的可靠性差距——格式合法不等于执行成功，偶尔成功与始终成功之间存在本质差异。COS-PLAY、GTA-2、SkillFlow 等工作分别从技能演化、工具使用和工作流生成等维度拓展智能体的能力边界。
